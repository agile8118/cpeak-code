# Full Code for the Cpeak Playlist

This repository contains all the code written for the Cpeak | Journey to Infinity! [YouTube Playlist](https://youtube.com/playlist?list=PLCiGw8i6NhvqsA-ZZcChJ0kaHZ3hcIVdY&si=xIOyVIid88WjbcDp).

## How to Use the Project Files

Each video in this series has its own dedicated folder. Inside each folder, you will find two versions of the code:

- **Starter:** The starter code needed to follow along with the video.
- **Finish:** The completed source code of the video.
